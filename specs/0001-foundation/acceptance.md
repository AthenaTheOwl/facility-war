# Spec 0001 — Foundation acceptance

## What "v0 done" means

Spec 0001 is closed when:

1. The three PRs in `tasks.md` are merged.
2. `uv pip install -e .[dev]` succeeds on a fresh venv.
3. `python -m facility_war --help` prints the CLI surface.
4. All local gates pass against checked-in fixtures.
5. `reports/2026-Q3-h100-substrate-shock/report.md` exists, was
   generated by the actual simulator (not hand-written), and passes
   voice_lint.
6. The COVID-2020 backtest runs and reports a Brier score under the
   threshold set in `eval/calibration_brier.py`.

## Commands to run

```bash
uv pip install -e .[dev]
python -m facility_war --help

uv run pytest
python scripts/voice_lint.py
python scripts/validate_schemas.py

python -m facility_war run \
  --graph data/graphs/h100_bom.json \
  --scenario scenarios/taiwan_substrate_90d.yaml \
  --trials 1000 \
  --seed 42 \
  --out reports/2026-Q3-h100-substrate-shock/

python eval/backtest_2020_covid.py
python eval/calibration_brier.py --max-brier 0.25
```

## Gates

| Gate | Source | Blocks merge when |
|---|---|---|
| pytest | `tests/` | any test fails |
| determinism | `tests/test_monte_carlo_determinism.py` | same seed produces different output |
| voice_lint | `scripts/voice_lint.py` | banned term in `reports/` |
| schemas | `scripts/validate_schemas.py` | any graph, scenario, or sim_run fails |
| calibration | `eval/calibration_brier.py` | Brier on COVID-2020 backtest above ceiling |

## What v0 explicitly does NOT include

- Customer-confidential supplier graphs (public fixtures only).
- Cost-optimization across mitigations.
- Recovery curves beyond end-of-scenario.
- Hosted SaaS or scheduled runs.
